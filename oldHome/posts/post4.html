8th December, 2020
In my very first blog post, entitled "Hello World", I noted the reasons why I wanted to create a blogging platform of my own. In particular, I felt that the existing plaforms I could use via npm, or another third-party company were quite heavy, and required quite a bit of work each time I wanted to create a new blog. In that spirit, I have written blog.js, a completely client-side blogging platform that allows a user to simply drag and drop a folder into their website project, and they now have the exact same platform that you are reading this post on.

Here is the link, if you want to check it out for yourself!

Now, I'd like to walk you through how to use the platform in it's entirety. While it's described in the blog.js README, I will also provide that information here. If you have any questions or comments, feel free to send me a message!

Getting Started

  1. in your project folder, run: git clone https://github.com/hguha/blog.js.git
  2. Open your project in a server. Any server will do.
  3. Profit

To Edit Template

The template, blog.html contains a "TO CUSTOMIZE" section that you can fill in. Once it's filled in, the blog is ready to use. Here are the option parameters that you can specify:
Name Description
title The title of the page and of the blog
subtitle The subtitle of the blog that appears in the sidebar
filters Dictionary that maps a filterName : filterLabel
postsFolder name of the folder that contains all posts
postPrefix prefix of each post's name that will be enumerated(e.g. post1, post2, etc. has a prefix of "post" and blog1, blog2, etc. has a prefix of "blog")


To Add/Edit Posts

Each post must be within the postsFolder and named as postPrefix followed by the next number(starting at 1). It will be a .html file. Then within your post file(the sample provided is post1.html), the template is as follows: Then within your post file(the sample provided is post1.html), the template is as follows:

<body>
  <div id="title">Title of Blog 1</div>
  <div id="tag">tutorial</div>
  <div id="date">22nd December, 2020</div>
  <div id="body">
    This is a sample blog post, written with raw HTML to your hearts desire.
  </div>
</body>

Each of the div-tag ids(title, tag, date, body) are necessary, along with the <body> tag and cannot be changed without altering the code in this version(future versions may allow any tag as a parameter to the blog object).

To Style

I like to think my styles are pretty clean, but if you want to change them, you can add/edit the styles directly within blog.css, or add your own stylesheet that is linked to blog.html.

I told you, insanely customizable!
