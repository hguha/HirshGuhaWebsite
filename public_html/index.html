Rubiks Cube

JAVASCRIPT • CSS • HTML
My take on a functioning Rubiks Cube model, written entirely in pure javascript, using CSS shaders controlled with a mouse
Rotation of the entire cube is based on a series of pivot points for the X, Y, or Z axes
Swiping on a given face rotates based on mouse movement in the clockwise or counterclockwise position
Auto solve and auto scamble features located in the top right hand corner, with control for ammount of turns

Sudoku Solver

HASKELL
Deterministic non-polynomial time sudoku solver written in haskell
unsolved puzzle can be pipelined in via a seperate text file, or inputted directly into the program
Handles non-unique cases by returning the first of many possibilities
Though the algorithm efficiency is non-polynomial, efficiencies mean it solves in under a second

Philosophy
